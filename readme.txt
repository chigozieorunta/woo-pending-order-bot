# WooCommerce Pending Order Reminder Bot

Send SMS messages to buyers notifying them of abandoned cart orders. It makes use of your Twilio API account to send SMS messages to your customers.

== Description ==

The WooCommerce Pending Order Reminder Bot plugin is a simple plugin that uses your Twilio account to send SMS messages to buyers notifying them of abandoned cart orders.

At a glance, this plugin helps you:

* Configure your Twilio ID and phone number.
* Configure the message you'd like to send customers notifying them of abandoned cart orders.

== Frequently Asked Questions ==

= What is my Twilio ID? =

Your Twilio ID is your registered ID with Twilio. You can get an ID by simply signing up on Twilio.

= What is my Twilio Phone Number? =

Your Twilio phone number is your registered phone number assigned to you by Twilio. This phone number works hand in hand with the Twilio API to enable you send messages easily to your customers.
